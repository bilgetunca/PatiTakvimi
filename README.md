# PatiTakvimi 🐾

SwiftUI ile geliştirilmiş bir evcil hayvan bakım takvimi ve hatırlatıcı uygulaması.

## Özellikler
- Etkinlik/hatırlatıcı ekleme, düzenleme, silme
- Mini takvim üzerinden gün bazlı görüntüleme
- Local Notifications ile bildirim
- Evcil hayvan profili / detay ekranları

## Teknolojiler
- Swift / SwiftUI
- UserNotifications
- Core Data (Persistence)

## Kurulum
1. Repo’yu klonlayın
2. Xcode ile açın: `PatiTakvim.xcodeproj`
3. Run
